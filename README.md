# Data Analytics Dashboard Project
This project showcases a **Power BI dashboard** analyzing survey data related to data professionals.

## Dataset
The dataset includes survey responses with columns like:
- Role, Salary, Industry
- Programming Languages
- Job satisfaction metrics (Salary, Work/Life Balance, Coworkers, Management, Upward Mobility, Learning New Things)
- Demographics (Gender, Age, Country, Education, Ethnicity)

## Power BI Dashboard
- Interactive visualizations of survey responses
- Analysis of career paths into data, satisfaction levels, and salaries
- Insights on programming language popularity and industry trends
